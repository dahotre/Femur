h1. Femur

Because I used Backbone.js to build this website, the name of another bone from the human body seemed like the right choice to name this app. Hence femur, the strongest bone in our body!

h2. Technology

This application requires:

* Ruby version 1.9.3
* "Rails version 3.2.13":http://railsapps.github.com/installing-rails.html
* Backbone.js
* MongoDB version 2.4.2
